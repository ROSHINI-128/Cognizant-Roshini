Bootstrap Accordion

Accordion Example

Click on the accordion headers to expand or collapse their content panels.

This is the first item's accordion body. It is shown by default, until the collapse plugin adds the appropriate classes that we use to style each element. These classes control the overall appearance, as well as the showing and hiding via CSS transitions. You can modify any of this with custom CSS or overriding our default variables. It's also worth noting that just about any HTML can go within the .accordion-body, though the transition does limit overflow.

This is the second item's accordion body. It is hidden by default. If you open this item, the first item will collapse. This is because both items share the same data-bs-parent="#myAccordion", making them part of the same accordion group.

This is the third item's accordion body. It is also hidden by default. You can put various types of content here, including text, images, lists, or even other Bootstrap components.
